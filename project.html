    CSS Summmary

    CSS (Cascading Style sheets) is used for styling your page content.

  • Styles are assigned via property-value pairs
  • You can assign via the "style" attribute
  • To avoid code duplication,you typically use global styles (eg via the "style" element) instead
  • Alternatively,you can work with external stylesheets files which you "link" to
  • When working CSS,concepts like "inheritance","specifity" and the "box model" should be understood

    • Learn more about all availabe CSS properties and values on the MDN CSS property refernce
